# Agenda Frontend

Frontend da agenda de contatos em Next.js.

## Desenvolvimento

Execute o servidor local:

```bash
npm run dev
```

Acesse [http://localhost:3002](http://localhost:3002) no navegador.

## Comandos

```bash
npm run lint
npm run build
```

O backend local esperado fica em `http://localhost:5080`, salvo quando `NEXT_PUBLIC_API_URL` estiver configurado.
